### Summary
Partition P5 (cross-cutting infra, DB, queue, v1 API) is mostly clean: no TODO/FIXME/HACK/XXX markers, no deprecated APIs, no commented-out code, and no stale console.log calls. The main debt categories are hardcoded localhost/ports in dev defaults, a handful of uncommented magic numbers in connection pool and queue configuration, a stale barrel file, and one operational gap where the queue service reads `DATABASE_URL` instead of the preferred `DATABASE_APP_URL`.

### Recommendations
1. **Queue service connection string (P1, ~15 min)** — Make `queue.service.ts` read `DATABASE_APP_URL` first and fall back to `DATABASE_URL`, mirroring `drizzle.provider.ts`. If `DATABASE_APP_URL` is intentionally skipped for pg-boss, add a comment explaining why (e.g. "pg-boss needs owner connection to create its own schema").
2. **Replace console.error with NestJS logger (P2, ~10 min)** — Inject `Logger` into `env.validation.ts` or return a warning object so the caller can log via the framework logger.
3. **Document pool constants (P2, ~10 min)** — Add JSDoc comments above `postgres(connectionString, { max: 10, idle_timeout: 20, connect_timeout: 10 })` explaining the rationale (e.g. "10 connections = 2x expected concurrent request burst").
4. **Document queue constants (P2, ~10 min)** — Add JSDoc to `QUEUE_DEFAULTS` explaining each field (e.g. `retryDelay: 30` = seconds).
5. **Delete stale barrel file (P3, ~5 min)** — Remove `common/index.ts` or add re-exports that are actually consumed by other modules.
6. **Add typed return interface for members list (P3, ~20 min)** — Replace the inline mapping in `v1Members.controller.ts` with a typed interface from `adminMembers.service.ts` so the comment at lines 65-66 can be removed.
7. **Fix overly broad type (P3, ~5 min)** — Change `expiresAt: string \| null` to `expiresAt: string` in `V1InvitationResponse`.
8. **Extract fallback retry-after constant (P3, ~5 min)** — Replace the magic `60` in `throttlerException.filter.ts` with a named constant like `FALLBACK_RETRY_AFTER_SECONDS`.
